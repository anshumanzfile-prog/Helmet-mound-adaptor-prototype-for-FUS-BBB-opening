# Helmet-Mount Adaptor Prototype for Focused Ultrasound (FUS) BBB Opening

## Overview
This repository contains the CAD files for a custom helmet-mount adaptor designed to improve positioning accuracy and ease of use for a low-cost single-element focused ultrasound transducer in small-animal (mouse) blood-brain barrier (BBB) opening experiments.

The design replaces the traditional stereotactic frame setup, which was found to be time-consuming and less precise. It consists of:
- Inner ring (holds the mouse head)
- Outer ring (holds the transducer and allows sagittal movement + rotation)

**Project:** Low-Cost Ultrasound Transducer for Ultrasound Therapy (BMED4010 FYP 2026)  
**Student:** Mall Anshuman (University No. 3036063797)

## Files Included
- `Outer_Ring.STL` – Outer ring of the helmet
- `Inner_Ring.STL` – Inner ring for mouse head

## Design Features
- Quick attachment via neodymium magnets
- Rotational freedom similar to CT-scan movement
- Pin-lock mechanism for angle adjustment
- Designed for easy 3D printing in PLA
